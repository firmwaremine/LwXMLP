This API is invoking the propore encoding converter function received by objenumEncodingTypeType, so the received protion will be converted from its encoding type to UTF-8 type.

Parameters
[in]objenumEncodingTypeType: the current encoding type of the data inside the pu8InputBuffer
[in]pu8InputBuffer: pointer to the data to be encoded
[in,out]pu32InputBufferLength: the length of the data to be encodded and it will be used to return the the amount of the encodded length out of the actual received length
[out]pu8OutputBuffer: buffer where the output encodded data will be stored in
[out]pu32OutputBufferLength: the encoodded data length
Returns
E_NOT_OK: if the encodding process is failed E_OK : if the encodding process is succedded
Note
the XML parser enigne shall be continusly invoked till parsing done or the parsing encountered an error
